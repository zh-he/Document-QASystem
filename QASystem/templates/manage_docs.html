正则表达式使用说明

使用说明：

在此输入的正则表达式将用于删除文档中的匹配内容。每行一个正则表达式，系统会按顺序处理。

格式要求：

  • 每行输入一个正则表达式模式
  • 系统会依次应用每个正则表达式，删除匹配的内容
  • 不需要添加前后的斜杠或修饰符

示例：

\d{6,}         # 删除6位及以上的数字
第\d+章        # 删除"第X章"格式的文本
\[\d+\]       # 删除[数字]格式的引用
\s{3,}        # 删除3个及以上连续空格

注：上面的注释仅作说明，实际输入时不要包含"#"及其后内容
